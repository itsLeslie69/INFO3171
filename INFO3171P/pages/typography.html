Examples

Wonderful Section Title: h3

Helpful h4's can show an example

Some interior text will be set to have a width of 90% of its parent div container. Using flexbox, the overall layout of this section is established. Flexbox is also used again in this div, set to column, and centered on the cross-axis to offset this example text.

Code

The Tailwind CSS library allows us to effectively style our elements.

Tailwind's width utility class is also used to control the width of this paragraph so it wraps neatly.

class="w-[90%]"

We often use padding to make more effective use of whitespace alongside the strong tag.

td, tr { padding: 2%; }

The base font size is 1rem - 16px - and we scale it from there, derived from the Tailwind default.

h4 { font-size: 1.5rem }
p { font-size: 1rem; }

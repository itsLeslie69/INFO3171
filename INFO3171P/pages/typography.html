Code

The Tailwind CSS library in combination with vanilla CSS allows us to effectively style our elements with a number of utilities and properties used. 

                        
h1, h2, h3, h4, a, p {
    font-family: Verdana;
}

h1 {
    font-size: 3rem /* 48px */;
    line-height: 1;
}

h2 {
    font-size: 2.5rem;
    line-height: 1;
}

h3 {
    font-size: 2rem;
    line-height: 1;
}

h4 {
    font-size: 1.5rem
}

p, a {
    font-size: 1rem;
}

Sometimes, the width of paragraph tags need to be set to 90% to ensure it does not touch other elements - either because of how it wraps or simply because there is nothing to separate it from its adjacent elements. 

                        
<p class="w-[90%]">
    Some interior text will be set to have a width of 90% of its parent div container.
</p>

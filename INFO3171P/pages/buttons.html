Buttons for Data Dashboard

Buttons on the data dashboard should serve clear purposes, enabling users to quickly perform actions. Below are the various button types used in the Star Air dashboard, categorized by function and state.

Priority Buttons

These buttons indicate different levels of action priority, such as primary actions, warnings, and general actions. They help users navigate the dashboard based on task importance.

<button class="bg-[#28A745] text-white font-bold py-3 px-6 rounded shadow-lg hover:bg-[#22963d] transition duration-200 ease-in-out transform hover:scale-105 cursor-pointer">Success</button>
<button class="bg-[#F4C96B] text-white font-bold py-3 px-6 rounded shadow-lg hover:bg-[#e3b55a] transition duration-200 ease-in-out transform hover:scale-105 cursor-pointer">Warning</button>
<button class="bg-[#BD0202] text-white font-bold py-3 px-6 rounded shadow-lg hover:bg-[#9d0202] transition duration-200 ease-in-out transform hover:scale-105 cursor-pointer">Error</button>
<button class="bg-[#83ABD0] text-white font-bold py-3 px-6 rounded shadow-lg hover:bg-[#6a9bbf] transition duration-200 ease-in-out transform hover:scale-105 cursor-pointer">Information</button>

Disabled Buttons

Disabled buttons are used when an action is not yet available or the user is not authorized to perform it.

<button class="bg-[#BBA79C] text-white font-bold py-3 px-6 rounded opacity-50 cursor-not-allowed" disabled>Primary Disabled</button>
<button class="bg-[#BBA79C] text-white font-bold py-3 px-6 rounded opacity-50 cursor-not-allowed" disabled>Secondary Disabled</button>
<button class="bg-[#BBA79C] text-white font-bold py-3 px-6 rounded opacity-50 cursor-not-allowed" disabled>Tertiary Disabled</button>

Navigation Structure

This section describes the navigation structure within the application, detailing the purpose and usage of each navigation element.

Overview Local Navigation Dropdown Navigation

Overview

The navigation menu allows users to easily access different sections of the application. It is designed to be intuitive and user-friendly.

<section class="flex flex-col gap-y-5">
  <h3 class="text-2xl">Overview</h3>

Local Navigation

Local navigation is used for jumping between related pages. For example, the account section includes links to Overview, Settings, and Billing.

Overview Settings Billing Integrations

Local navigation is present on all pages under /account and indicates the currently viewed section in the specified color.

<nav class="flex flex-col gap-y-5">
  <h3>Overview</h3>
  <ul>
    <li><a href="#/account/">Overview</a></li>
    <li><a href="#/account/settings">Settings</a></li>
    <li><a href="#/account/billing">Billing</a></li>
    <li><a href="#/account/integrations">Integrations</a></li>
  </ul>
</nav>

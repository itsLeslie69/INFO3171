Chart Design Guidelines

Our charts should always prioritize clarity and simplicity. Each chart type serves a specific purpose and follows a set of design rules to maintain consistency and avoid unnecessary complexity.

Line Charts

Line charts are used for visualizing trends over time. Ensure the line is smooth, 2px thick, and never curved unless required. Dotted lines can be used to represent secondary metrics such as averages or projections. Dots may be added at key data points to provide extra clarity.

Bar Charts

Bar charts should be used to compare different categories or values. The spacing between bars should always be 50% of the bar width. If using stacked bars, opt for distinct color combinations to enhance readability.

Stacked Bar Charts

Stacked bar charts are an excellent choice for visualizing how individual categories contribute to the whole. Choose contrasting colors for each category to maintain clarity. Make sure to avoid overlap when possible.

Scatter Plots

Scatter plots are ideal for showing relationships between variables. The points should be 8px by 8px, and if multiple categories are used, vary the shape (circle, square, triangle) to aid differentiation. Ensure that colors from our palette are used to maintain brand consistency.

Donut Charts

Donut charts can be used to represent parts of a whole, with a central hole to add emphasis on the data proportions. Use strong contrast between segments to ensure the data is easily distinguishable.

Half Donut Charts

Half-donut charts are an effective way to display data in a visually impactful manner. They allow for proportional data to be showcased without overcrowding the chart.

Pie Charts

Pie charts can be used to display proportions. Each section should be colored distinctly to avoid confusion, and labels should be clear and readable.

Single Value Charts

Single value charts, such as gauges, help display key metrics quickly and easily. Ensure that the main value is easily readable, with appropriate colors for positive or negative outcomes.

Area Charts

Area charts are used to represent data over time, similar to line charts but with shaded areas beneath the lines. Use gradients or solid fills to indicate volume.

Horizontal Bar Charts

Horizontal bar charts are effective when comparing categories with longer names or more complex datasets. Use consistent spacing and clear labels.

Progression Charts

Progression charts are useful to visualize ongoing progress towards a goal. These charts are represented as bars or meters, with clear markers indicating milestones.

Color Palette for Charts

Grid Lines and Axes

Grid lines should always support the data without drawing too much attention. Use 2px axis lines and grid lines with 50% opacity to create a balance. Keep x-axis labels evenly spaced, and reduce label crowding by adjusting ticks as needed.

Fly-overs

Fly-overs provide additional information for data points on hover. Ensure that fly-over pop-ups do not obscure the primary data and remain readable.
